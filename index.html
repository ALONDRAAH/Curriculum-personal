Una aplicación web para gestionar reservas de taxi en tiempo real, permitiendo a los usuarios realizar solicitudes, verificar disponibilidad y hacer pagos en línea.

  • Backend: Node.js, Express
  • Base de Datos: MongoDB
  • Características:
    • Interfaz intuitiva para usuarios y administradores.
    • Integración de pagos con API de Stripe.
    • Geolocalización de taxis con API de Google Maps.
